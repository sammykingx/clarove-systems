# Clarove Systems Limited Website

This project is a modern, responsive landing page for **Clarove Systems Limited** — a software and digital solutions company focused on building reliable, scalable, and high-performance platforms for businesses.

![Clarove Systems mockup](/assets/images/mockups/clarove-mk-1.jpeg)

## What the project is about

The website is designed to:
- introduce Clarove Systems and its brand positioning
- highlight enterprise software capabilities and digital platform services
- showcase credibility through performance, security, and operational strength messaging
- encourage visitors to explore the portfolio or request a consultation

## Main purpose

The site acts as a professional online presence for the company, helping potential clients understand:
- what Clarove Systems does
- the kind of solutions it provides
- why the company is suited for mission-critical and growth-focused digital projects

## Features

- clean, modern one-page layout
- responsive design for desktop and mobile devices
- strong branding and visual hierarchy
- call-to-action sections for portfolio and consultation requests
- lightweight static site setup with Tailwind-based styling

## Tech stack

- HTML for page structure
- Tailwind CSS for styling
- JavaScript for interactive components
- Alpine.js for UI behavior

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development watch process:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project structure

- `index.html` — main page content
- `assets/css/` — stylesheet files
- `assets/js/` — JavaScript logic and reusable components
- `assets/images/` — icons, illustrations, and media assets

## Notes

This repository is primarily a static marketing website, so the main goal is presentation, brand storytelling, and generating interest from prospective clients or partners.
